# Sitio Web Institucional - Consultorio Odontológico

Este proyecto consiste en una plataforma web moderna y funcional diseñada para optimizar la presencia digital y la gestión de pacientes de un consultorio odontológico.

## 🚀 Características y Funcionalidades
- **Interfaz Responsiva:** Diseño adaptativo para una visualización óptima en dispositivos móviles y escritorio.
- **Gestión de Turnos Dinámica:** Integración de enlaces directos a WhatsApp para agilizar la comunicación y reserva de turnos.
- **Geolocalización:** Incorporación de Google Maps para facilitar la ubicación del consultorio.
- **Conectividad Social:** Enlaces integrados a Instagram y otras redes sociales institucionales.
- **Acceso a Obras Sociales:** Secciones con redirecciones directas para información de coberturas médicas.

## 🛠️ Tecnologías Utilizadas
- **Frontend:** HTML5, CSS3 y JavaScript.
- **Deployment:** Desplegado y hosteado en **Vercel**.
- **Integraciones:** API de WhatsApp y Google Maps.

## 📦 Instalación y Uso
1. Clona el repositorio:
   `git clone https://github.com/DelfinaBaravalle/baravalle-consultorio.git`
2. Abre el archivo `index.html` en tu navegador.
